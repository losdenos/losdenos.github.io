ABOUT THE PROJECT

A real-time household energy monitoring system built as a final year project in 2020. The device measures AC voltage and current using a ZMPT101B voltage sensor and SCT-013 current transformer, computes RMS values and power consumption on an ESP32, and displays readings on a 20×4 I²C LCD.

Data is pushed wirelessly over Wi-Fi to the Blynk IoT platform, where a custom mobile dashboard shows live voltage, current, wattage, and cumulative kWh — accessible from anywhere on a phone.
